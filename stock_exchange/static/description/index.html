Stock Exchange

Stock Exchange From Return of Stock.

INTRODUCTION

Stock Exchange module allows us to exchange the products form odoo backend which is already delivered and invoiced.

INSTALLATION

  • Can easily exchange from return wizard.
  • Can specify the exchange quantity..
  • Can specify the other products too.
  • Separate exchange orders.
  • Mapping between exchange orders and original sale orders through smart buttons.
  • Mapping between exchange pickings and original picking through smart buttons and widget.
  • Preparing the invoice, credit notes automatically.
